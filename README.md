# 📸 VisionFX

VisionFX is a feature-rich desktop camera application built with **Python** and **OpenCV**. It provides real-time camera filters, photo capture, video recording, countdown timer, microphone audio recording, and live FPS monitoring through an easy-to-use interface.

---

## ✨ Features

- 🎥 Real-time webcam preview
- 🎨 Multiple live camera filters
  - Blur
  - Cartoon
  - Grayscale
  - Negative
  - Sepia
  - Sharpen
  - Sketch
- 📷 Capture high-quality photos
- ⏳ Photo countdown timer
- 🎬 Record videos
- 🎤 Record microphone audio
- ⚡ Live FPS display
- 💾 Automatically saves photos and recordings

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/VisionFX.git
```

Navigate into the project

```bash
cd VisionFX
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```

---

## 📸 Available Filters

| Filter | Description |
|---------|-------------|
| Blur | Smooths the image |
| Cartoon | Cartoon-style effect |
| Grayscale | Converts image to black & white |
| Negative | Inverts image colors |
| Sepia | Vintage brown-tone effect |
| Sharpen | Enhances image details |
| Sketch | Pencil sketch effect |
